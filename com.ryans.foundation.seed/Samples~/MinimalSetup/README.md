# Minimal Setup Sample

1. Copy this package into `Packages/com.your.foundation.seed`.
2. Open Unity and run `Tools/Foundation Seed/Setup/Run Initial Setup`.
3. Confirm config assets were created under `Assets/Plant/Resources/`.
4. Enter Play Mode, then run `Tools/Foundation Seed/Diagnostics/Validate Foundation Setup`.
5. Use `FoundationSeedTimedBehaviour` or `IFoundationSeedTimedClient` for project time-driven systems.
